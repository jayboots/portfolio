Projects

Hotel Booking Website

Description

The hotel booking website was a term project for University of Ottawa course CSI 2132: Databases I. In this project, our team designed and deployed a CRUD-capable website to enable users to search for and reserve hotels, as well as for participating hotel clients to update and manage their hotel resources within the system.

Technologies

The web app itself was developed using react, with additional styling and features added in CSS, HTML, and JavaScript. CRUD interactions were scripted in PHP, while the back-end database used MySQL and the phpMyAdmin app as an interface to a locally-hosted Apache Server (XAMPP).

Weather Prediction Model

Description

A mid-term project for University of Ottawa course CSI 4106: Introduction to Artificial Intelligence involved the creation of model that could predict if it would rain on any given day based on observations about the weather and location.

The model has general applications in the field of meteorology and weather forcasting, and indirect applications in many other fields. For example, airlines must be aware of weather conditions before flying planes, farmers must sow their fields in time for rain, outdoor activities must be postponed due to bad weather, and so on.

Mathematical Methods

  • Naive Bayes
  • Logistic Regression
  • Multi-Layer Perceptron

Technologies

This project was developed exclusively in Python (pandas, numpy, seaborn, and matplotlib) using datasets from Kaggle.

Online Property Rental System

Description

This term project for the University of Ottawa course SEG 3102 Software Design and Architecture involved the development and deployment of an Online Property Rental System (OPRS) for the finctional real estate companty Houses of Dreams. The OPRS took the form of a web portal in which both employees of Houses of Dreams and members of the public could manage listing inventories and browse available listings, respectively

Technologies

We used a Fat-Client style architecture for our implementation. The client-side web portal was implemented in TypeScript using the Angular Framework. The server was implemented using Firebase for the express purpose of not implementing any SQL in this project.
